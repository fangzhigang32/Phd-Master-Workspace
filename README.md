## 📖 项目简介

PhD Master Workspace 是一款专为硕博生量身打造的桌面工作管理应用。它将科研进度追踪、任务管理、专注计时、投稿管理、健康管理等功能整合在一个界面中，帮助你高效管理硕博生涯的方方面面。

### 为什么需要这个工具？

- **科研进度可视化** - 硕/博士论文进度、投稿流程一目了然
- **时间管理一体化** - 打卡、专注计时、日程规划无缝衔接
- **心理健康关怀** - 内置心情记录、成就系统、复盘功能
- **数据本地存储** - 所有数据保存在本地，隐私安全有保障
- **多主题支持** - 9 种精心设计的主题，保护你的眼睛

## ✨ 功能特性

### 🏠 总览首页
- **打卡记录** - 支持一天多次打卡，自动计算工作时长
- **今日执行** - 整合待办任务，支持开始/结束自动生成专注记录
- **专注计时器** - 番茄钟式专注计时，支持手动补录
- **日程时间块** - 垂直时间轴视图，任务可视化排期

### 📊 项目看板
- **项目管理** - 创建长期项目，设置截止日期和阶段
- **任务拆解** - 将项目拆分为可执行任务
- **四象限分类** - 重要紧急矩阵，优先级一目了然
- **进度追踪** - 实时统计项目完成度

### 📚 硕/博士毕业论文进度
- **里程碑管理** - 设置论文关键节点（开题、预答辩、答辩等）
- **章节进度** - 逐章追踪写作进度（草稿/修改/完成）
- **推进日志** - 记录每日写作投入，支持字数统计
- **整体进度估算** - 自动计算论文整体完成度

### 📬 投稿管理
- **投稿项目追踪** - 管理论文投稿全流程
- **阶段流转** - 准备中 → 投稿中 → 审稿中 → 修改中 → 已接收/已拒稿
- **成果归档** - 已发表论文自动归档展示
- **推进日志** - 记录每次修改、审稿意见回复等

### 💪 健康管理
- **习惯打卡** - 设置每日习惯目标，追踪完成率
- **运动记录** - 记录运动类型和时长
- **睡眠追踪** - 记录睡眠时间和质量
- **心情记录** - 情绪追踪，关注心理健康

### 🌱 心灵关怀
- **每日心情** - 记录情绪状态
- **感恩日记** - 培养积极心态
- **激励语录** - 每日正能量

### 👨‍🏫 向上管理导师
- **沟通记录** - 记录与导师的每次交流
- **待办事项** - 导师交代的任务追踪
- **汇报准备** - 组会/周会汇报准备清单

### 📝 每日复盘
- **日总结** - 记录今日收获和不足
- **明日计划** - 规划第二天工作
- **周/月复盘** - 更大周期的回顾

### 🏆 成就殿堂
- **成就系统** - 解锁各种成就徽章
- **里程碑奖励** - 完成重要节点获得成就
- **数据统计** - 累计专注时长、完成任务数等

### 📈 数据看板
- **多维度统计** - 日/周/月数据汇总
- **图表可视化** - 专注时间趋势、任务完成率等
- **数据导出** - 支持 JSON 格式导出备份

### 🎨 多主题支持
提供 **9 种精心设计的主题**：
- 浅色默认 - 经典白色主题
- 深色模式 - 护眼暗色主题
- 暖色调 - 温暖的米黄色调
- 冷色调 - 清爽的蓝灰色调
- 护眼绿 - 经典护眼绿色
- 专注蓝 - 提升专注力的蓝色
- 日落橙 - 温暖的橙色调
- 深海蓝 - 深邃的海洋蓝
- 森林绿 - 沉静的森林绿

## 🚀 下载安装

1. 前往 [Releases](https://github.com/fangzhigang32/Phd-Master-Workspace/releases/tag/release) 页面
2. 下载最新版本的 `PhDWorkspace-Portable.exe`
3. 双击安装，选择安装位置（记住安装路径）
4. 安装完成后，找到`安装路径`下的文件`PhD Master Workspace.exe`，双击运行（可鼠标右击选择`固定到开始屏幕`或`固定到任务栏`）


### 系统要求

- **操作系统**: Windows 10 / 11 (64位)
- **磁盘空间**: 约 200MB

## ❓ 常见问题

### Q: 打包后的 exe 可以在没有 Node.js 的电脑上运行吗？

**A**: 可以！打包后的 exe 是完全独立的，内置 Chromium 浏览器内核，不依赖任何外部运行时。

### Q: 数据会丢失吗？

**A**: 数据保存在本地存储中，只要不清理应用数据就不会丢失。建议定期导出备份。

### Q: 支持其他操作系统吗？

**A**: 目前仅支持 Windows。如需支持 macOS/Linux，可修改 `package.json` 中的打包配置。

### Q: 打包体积为什么这么大？

**A**: 约 72.5MB，因为包含了完整的 Chromium 浏览器内核。
