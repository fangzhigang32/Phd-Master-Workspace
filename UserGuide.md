## 1. 总览首页

> 把打卡、任务、专注和今日日程放在同一屏，适合日常快速推进。

### 1.1 打卡记录

**功能说明**：记录工作开始/结束时间，支持请假登记。

**操作步骤**：
1. 点击 **「开始工作/到位打卡」** 记录上班时间
2. 点击 **「结束工作/离开打卡」** 记录下班时间
3. 选择请假类型后点击 **「请假」** 登记请假

**请假类型**：病假、事假、会议、出差、休息、其他

**统计显示**：
- 今日打卡次数
- 今日工作时长
- 今日请假记录
- 未结束工作段

**快捷操作**：
- 关闭今天所有未结束工作段
- 清空今日本日请假记录

### 1.2 今日执行

**功能说明**：整合今日候选任务、到期任务和当前进行中任务。

**操作步骤**：
1. 在输入框输入临时任务名称
2. 点击 **「新增临时任务」** 添加任务
3. 点击任务卡片上的 **「开始」** 开始执行
4. 完成后点击 **「完成」** 或 **「放弃」**

**自动关联**：开始/结束任务会自动生成专注记录和日程时间块。

### 1.3 专注计时器

**功能说明**：番茄钟式专注计时，支持分类记录。

**操作步骤**：
1. 输入专注主题（可留空，自动读取当前任务）
2. 选择分类：科研/写作/阅读/事务/其他
3. 可选填写备注
4. 点击 **「开始专注」** 开始计时
5. 完成后点击 **「结束专注」** 或 **「放弃本次」**

**手动补录**：
- 选择日期、输入主题、开始时间、结束时间
- 点击 **「添加专注记录」**

### 1.4 今日日程

**功能说明**：垂直时间轴展示今日安排，支持时间块规划。

**时间块规划**：
1. 从「今日执行」下拉选择任务
2. 设置开始时间和结束时间
3. 可修改时间块标题
4. 点击 **「排入日程」**

**视图切换**：可切换查看其他日期的日程。

## 2. 项目看板

> 先管理项目，再拆解任务；项目看板只负责规划，总览首页负责具体执行。

### 2.1 项目管理

**新增项目**：
1. 点击 **「新增项目」** 按钮
2. 填写项目信息：
   - 项目名称（必填）
   - 预期结果
   - 所属领域：科研/写作/投稿/项目/生活/其他
   - 状态：进行中/暂停/已完成
   - 开始日期
   - 截止日期
   - 备注

**项目列表操作**：
- 点击项目行可在任务表中筛选该项目
- 编辑项目信息
- 查看项目推进日志

### 2.2 任务管理

**任务属性**：
- 标题（必填）
- 所属项目
- GTD分类：收集箱/下一步/等待反馈/将来也许/已完成
- 四象限：重要且紧急(Q1)/重要不紧急(Q2)/紧急不重要(Q3)/不紧急不重要(Q4)
- 今日安排：今日必做/今日应该/今日可以
- 状态：计划中/没开始/进行中/完成
- 截止日期
- 预估时长
- 上下文
- 备注

**筛选功能**：
- 显示全部
- 只看今日候选
- 按四象限筛选（Q1/Q2/Q3/Q4）
- 按项目筛选

**新增任务**：点击 **「新增任务」** 按钮，填写任务信息。

## 3. 硕博毕业论文进度

> 用里程碑 + 章节进度 + 推进日志，把论文推进做成可统计的看板。

### 3.1 论文信息

**可填写内容**：
- 论文题目
- 目标答辩日期
- 当前版本
- 备注

点击 **「保存信息」** 后生效。

### 3.2 里程碑管理

**默认里程碑**：
- 开题/Proposal
- 中期检查
- 预答辩
- 论文提交
- 正式答辩

**操作**：
- 勾选完成里程碑
- 设置截止日期
- 添加自定义里程碑

### 3.3 章节进度

**默认章节**：
- 引言/Introduction
- 相关工作/Related Work
- 方法/Method
- 实验/Experiments
- 结论/Conclusion

**操作**：
- 拖动进度条更新进度（0-100%）
- 设置状态：草稿/修改/完成
- 添加自定义章节

### 3.4 论文推进日志

**记录内容**：
- 日期
- 类型：写作/修改/实验/讨论组会/其他
- 投入分钟数
- 字数（可选）
- 备注

## 4. 投稿管理

> 保留投稿流程核心能力，用看板方式管理投稿项目。

### 4.1 投稿流程看板

**流程阶段**：
1. 选题中
2. 写作中
3. 待投稿
4. 已投稿
5. 审稿中
6. 返修中
7. 已接收
8. 已见刊/已收录
9. 搁置/拒稿

**操作**：拖拽卡片在不同阶段间移动。

### 4.2 新增投稿项目

**填写信息**：
- 题目/项目名
- 期刊/会议名
- 截稿日期
- 当前阶段
- 类型：Journal/Conference/Workshop/Other
- 备注

### 4.3 投稿推进日志

每个投稿项目都可以记录推进过程：
- 选择投稿项目
- 日期
- 类型：写作/修改/实验分析/投稿材料/沟通/推进
- 投入分钟
- 详细记录

**导出**：可下载为 Markdown 文件。

## 5. 健康管理

> 把睡眠、运动、饮食等健康习惯集中管理。

### 5.1 健康习惯记录

**默认习惯**：
| 习惯 | 图标 | 记录方式 |
|------|------|----------|
| 早睡 | 🌙 | 时间记录 |
| 早起 | 🌞 | 时间记录 |
| 运动 | 🏃 | 时长记录 |
| 饮食记录 | 🍽️ | 饮食专用 |

**记录方式说明**：
- **时间记录**：记录具体时间点（如22:30睡觉）
- **时长记录**：记录持续时长（如运动45分钟）
- **勾选完成**：简单打勾标记
- **次数记录**：记录次数
- **文字记录**：自由文本描述

### 5.2 饮食记录

**操作**：
1. 选择餐次：早餐/午餐/晚餐/加餐/饮水饮品
2. 输入记录内容
3. 点击 **「添加饮食记录」**

### 5.3 体重记录

**操作**：
1. 输入数值
2. 选择单位：kg/斤/lb
3. 点击 **「添加体重记录」**

### 5.4 习惯管理

**添加自定义习惯**：
1. 输入习惯名称
2. 输入图标（如：💧）
3. 选择记录方式
4. 点击 **「添加习惯」**

**管理操作**：修改名称/图标/记录方式，或停用/删除习惯。

## 6. 心灵关怀

> 把硕博阶段常见的压力、能量波动、自我关怀放到一个温和的记录页。

### 6.1 今日心理健康记录

**此刻状态**：
- 选择心情表情：😣😕😐🙂😊

- 等待什么

## 7. 向上管理导师

> 把汇报、等待反馈、导师承诺和下一步写清楚，适合导师容易忘记时使用。

### 7.1 今日导师沟通记录

**沟通状态**：
- 当前状态：准备汇报/已汇报/已沟通/等待反馈/需要主动推进
- 沟通方式：面谈/邮件微信/文稿批注/组会/其他
- 沟通压力：1-5
- 预期清晰度：1-5

**记录内容**：
- 这次沟通主题
- 我已经准备好的进展/证据
- 我希望导师给的支持/决策
- 当前卡点/误解风险
- 导师这次的反馈/决策

### 7.2 防忘管理

**核心功能**：记录导师说过的话/承诺/计划，避免只靠记忆承担风险。

**记录内容**：
- 导师说过的话/承诺/计划
- 我复述确认/会后纪要
- 下次核对日期
- 追踪状态：待核对/已确认待兑现/需再次提醒/已落实

### 7.3 待追踪承诺

自动展示所有未落实的导师承诺，到核对日期会自动提醒。

## 8. 每日复盘

> 用PhD学术复盘模板记录具体产出、未竟分析、学术洞见。

### 8.1 PhD学术复盘

**状态/能量值**：
- ☀️ 高：可以攻坚
- 😐 中：稳定推进
- 🌧️ 低：需要降载

**复盘内容**：

- ⏳ 未完成与拖延分析：卡在哪里？
- 💡 学术洞见与新发现：学到的新方法/观点

### 8.2 明日优先任务

设置明日 Priority 1/2/3 任务，保存后自动加入任务总表，截止日期设为明天。

### 8.3 当天记录统计

自动汇总当天所有模块的关键记录，可下载为 Markdown。

## 11. 数据管理

> 导入导出、备份恢复、数据概览。

### 11.1 导出备份

**方式一**：点击 **「下载 JSON 备份」**，保存为文件。

**方式二**：点击 **「复制 JSON 到剪贴板」**，粘贴到安全位置。

### 11.2 导入恢复

**方式一**：点击 **「导入 JSON 文件」**，选择备份文件。

**方式二**：粘贴 JSON 到文本框，点击 **「导入这段 JSON」**。

### 11.3 清空数据

点击 **「清空全部数据」**，会弹出确认提示，防止误操作。

⚠️ **重要**：清空前请先导出备份！

## 数据备份与恢复

### 数据备份

应用数据保存在本地，建议定期备份：

- **导出**: 菜单栏 → 文件 → 导出数据备份 (Ctrl+S)
- **导入**: 菜单栏 → 文件 → 导入数据备份 (Ctrl+O)

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 导出数据备份 |
| `Ctrl+O` | 导入数据备份 |
| `Ctrl+R` | 重新加载 |
| `F11` | 全屏切换 |
| `Ctrl+Q` | 退出应用 |

---


### 数据存储位置

- **便携版**: 数据存储在 exe 文件同目录下的 `PhD Master Workspace` 文件夹中


### 数据格式

所有数据以 JSON 格式存储，包括：
- 打卡记录
- 任务和项目
- 论文进度
- 投稿信息
- 习惯和健康数据
- 成就记录

### 数据迁移

1. 在原设备导出数据备份 (Ctrl+S)
2. 将 JSON 文件复制到新设备
3. 在新设备导入数据备份 (Ctrl+O)

## 常见问题

### Q1: 数据会丢失吗？

**A**: 数据保存在浏览器本地存储中，正常使用不会丢失。但仍然建议定期导出备份。

### Q2: 数据可以同步到其他设备吗？

**A**: 目前不支持自动同步。需要手动导出备份，在另一台设备上导入。


### Q3: 主题设置会保存吗？

**A**: 会。主题设置保存在浏览器本地存储中，下次打开会自动应用。
